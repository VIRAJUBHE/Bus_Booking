# 🚌 Bus Booking System

## 📌 Overview
A full-stack application that allows users to search buses, select seats, and book tickets in real-time with accurate availability tracking and secure booking flow.

## 🚀 Features
- Search buses by source, destination, and date  
- Real-time seat selection with visual layout  
- Prevent duplicate bookings using backend validation  
- Booking confirmation system  
- Date-based validation for valid bookings  
- Responsive UI with loading states  

## 🛠 Tech Stack
Frontend: Angular, TypeScript, HTML, CSS  
Backend: Spring Boot, Java, REST APIs  
Database: MySQL  

## 🏗 Architecture
- Angular frontend interacts with Spring Boot backend via REST APIs  
- Backend manages booking logic and validations  
- MySQL stores bus, seat, and booking data  
- Angular updates UI dynamically
